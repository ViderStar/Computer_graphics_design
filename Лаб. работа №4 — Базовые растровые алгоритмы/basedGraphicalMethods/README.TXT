ЦЕЛЬ РАБОТЫ:
Написать приложение, иллюстрирующее работу базовых растровых алгоритмов:
− пошаговый алгоритм;
− алгоритм ЦДА;
− алгоритм Брезенхема;
− алгоритм Брезенхема (окружность).

ЗАДАЧИ РАБОТЫ:

СРЕДСТВА РАЗРАБОТКИ:
Язык программирования Python 3.10 + библиотека matplotib + библиотека time

ХОД РАБОТЫ:
В окне приложения изображены график каждого алгоритма (всего 4). Внизу есть поле ввода
Ввод осуществляется в следующем формате: координаты X иY первой точки, координаты X и Y второй точки.
В случае с алгоритмом Брезенхема для окружности первая точка является центром окружности, а вторая точка лежит на окружности.
Графики строятся по точкам, каждая точка равняется одному пикселю, поэтому желательно работать с большими значениями.

ВЫВОДЫ:
Написал приложение, иллюстрирующее работу базовых растровых алгоритмов, используя две библиотеки на языке Python.
